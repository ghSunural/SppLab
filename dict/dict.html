 1) Определение параметров геофизич. полей по заданным характеристикам источника. Каждый из крупных разделов современной геофизики имеет свои задачи в соответствии со своим частным предметом исследования. Решение этих задач основывается на применении принципов и методов математич. физики. Прямые задачи геофизики служат основой для интерпретации геофизич. данных при решении обратных задач геофизики. 2) По известному объекту с заданными физическими свойствами найти поле. 3) Определение параметров формирующегося физического поля по известным параметрам модели изучаемой среды.
